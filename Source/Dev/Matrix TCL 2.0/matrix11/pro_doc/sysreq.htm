System Requirements

This class can be used on any operating system having an ANSI / ISO compatible C++ compiler. The C++ compiler must have good support for template, namespace, and valarray class. This class has been tested with the following compilers only.

  • MS Visual C++ 5.0, 6.0 & 7.0
  • Borland C++ 5.5
  • C++ Builder 4.0 & 5.0
  • Metrowerks CodeWarrior 6.0 or higher

However, this class can be used with any other modern C++ compilers on any platform. If you are using, or want to use a different compiler not listed above, please contact us at matrix@techsoftpl.com.

Installation

To install the Matrix TCL Pro, please copy "cmatrix", "matrix.h", and "matrix.cc" files into the INCLUDE directory of your compiler. Alternatively, you can add the full path name to these files into the INCLUDE path list of your compiler. To uninstall the library, remove these files from your computer.

Testing

To test the Matrix TCL Pro class, we have included a test program named testmat.cpp with this package. You may compile and run this program on your machine to test this class. This program should be compiled by your compiler error free. However, since most of the tests are performed on randomly generated matrices, some test may fail occasionally due to ill-conditioned / singular matrix, and/or for greater round-off error produced in some matrix operations. If you encounter any reproducible compile-time, or repeated run-time error, please send the complete error information along with the name and version number of your compiler to <matrix@techsoftpl.com>.
