JavaScript Table of Contents Generator (JSTOC)

These pages use frames and JavaScript. To view them, please follow these instructions:
  • Use a frame-capable browser that can execute JavaScript programs.
    For example, you can use
       Netscape Navigator* 3.0 or newer version, or
       Microsoft Internet Explorer* 3.0 or newer version.
  • Enable JavaScript in your browser:
    In Microsoft Internet Explorer, go to View | Options | Security and check Run ActiveX scripts.
    In Netscape Navigator, go to Options | Network Preferences | Languages and enable JavaScript.
    In Netscape Communicator, go to Edit | Preferences | Advanced and enable JavaScript.
  • If you did the above and still have problems, reloading the page may help.

The JSTOC script was tested with both Netscape Navigator 3.0 and Microsoft Internet Explorer 3.0 under Windows* 95 and Windows NT* 3.51 and 4.0. It may or may not work with older browsers or under other OS.
