LAPACK Routines

The Intel® Math Kernel Library includes LAPACK routines that perform the following groups of tasks:

  • Triangular factorization
  • Solving systems of linear equations
  • Solution refinement and error bounds for linear equations
  • Computing condition numbers of matrices
  • Matrix inversion
  • QR and LQ factorization
  • Computing singular value decomposition
  • Solving symmetric eigenvalue problems
  • Solving nonsymmetric eigenvalue problems
  • Reducing generalized eigenvalue problems to standard problems
  • Solving Sylvester's equation AX + XB = C
