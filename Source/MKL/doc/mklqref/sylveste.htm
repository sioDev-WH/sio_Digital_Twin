Sylvester's Equation

?trsyl solves Sylvester’s equation for real quasi-triangular or complex triangular matrices A and B:

op(A)X + isgn Xop(B) = scale C

where A is an m by m matrix,
B is an n by n matrix
C and X are m by n matrices,
isgn is either 1 or -1;
scale is a scale factor determined by the routine to avoid overflow in X. For the definition of op(A) and op(B), see Matrix Arguments.

call strsyl (trana, tranb, isgn, m, n, SA, lda, SB, ldb, SC, ldc, sscale, info)
call dtrsyl (trana, tranb, isgn, m, n, DA, lda, DB, ldb, DC, ldc, dscale, info)
call ctrsyl (trana, tranb, isgn, m, n, CA, lda, CB, ldb, CC, ldc, sscale, info)
call ztrsyl (trana, tranb, isgn, m, n, ZA, lda, ZB, ldb, ZC, ldc, dscale, info)
