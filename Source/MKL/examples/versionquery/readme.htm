This directory contains examples for using the MKLGetVersion and MKLGetVersionString functions. To run examples simply run the mklgetversionall.bat file. As a result, two *.sum files will be created. mkl32_version_c.sum file will contain the result of running "MKLGetVersion" functions in C context. mkl32_version_f.sum file will contain the result of working "MKLGetVersion" functions in Fortran context.

The *.sum files should contain the information like this when run correctly:
